## 개발환경 및 라이브러리
- 백엔드 : Java, Spring Boot , JSP
- 프론트 : HTML5, CSS, JS, jQuery, Vue3, axios,element-plus
- DB : Maria db, Mybatis
- WAS : Tomcat
- 서버 : AWS (Lightsail)
- 형상관리 : Git, GitHub
